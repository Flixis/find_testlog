Barcode Scanner Support Tutorial

To enable barcode scanner support, follow the steps below:

  1. Place the file J:\Service and Repair\Tools\LogFinder_tauri\find_testlog_tauri.xml inside the folder C:\Program Files\Prodrive\Scanner\Recipes.Local.
  2. Restart the scanner application by running scanner.exe from C:\Program Files\Prodrive\Scanner.exe.
  3. After these steps, the scanner should be configured to work with the new settings.
